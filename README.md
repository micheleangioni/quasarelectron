# Quasar Electron Skeleton

> A Quasar project to build Electron applications

## Build Setup

``` bash
# install dependencies
$ npm install
$ cd electron
$ npm install

# serve with hot reload at localhost:8080
$ quasar dev

# load Electron
$ cd electron
$ quasar dev
```
